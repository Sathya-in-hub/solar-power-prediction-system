# Solar Power Prediction ML Service 🤖☀️

Machine Learning service for predicting solar radiation intensity across different regions of India. This service is part of the "AI-Based Solar Power Prediction and Optimization System" project.

## 🎯 Features

- **Solar Intensity Prediction** - Predict kWh/m² for any date and region
- **Sunshine Hours Calculation** - Estimate daily sunshine duration
- **Suitability Score** - 0-100 rating for solar potential
- **Risk Assessment** - LOW/MODERATE/WARNING/CRITICAL levels
- **Battery Recommendations** - Smart storage suggestions
- **Batch Predictions** - Process multiple requests at once
